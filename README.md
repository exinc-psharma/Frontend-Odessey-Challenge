# Evolution of the Internet Storyteller

An interactive, scroll-driven storytelling experience traveling through the eras of the internet.

## Tech Stack
- **React (Vite)**
- **GSAP (ScrollTrigger)**
- **Framer Motion**
- **Lucide React**
- **Vanilla CSS**

## Sections
1. **Hero**: Immersive glitch intro.
2. **ARPANET**: Signature interactive network graph.
3. **Dot-Com**: 90s retro web windows.
4. **Social & Mobile**: Modern app-grid interface.
5. **Future**: Web3 & AI glassmorphism with 3D elements.

## Features
- Interactive Scroll-Linked Timeline.
- Cinematic Scroll Snapping.
- Smooth Theme Morphing.
- "Booting Up" Loading Animation.

## Setup
```bash
npm install
npm run dev
```
